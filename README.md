# Supply Chain Analytics Dashboard

A professional dashboard built using React + Vite + Streamlit concepts to analyze and predict supply chain data.

## Features
- KPI Metrics
- Distribution Analysis
- Correlation Heatmap
- Category Insights
- Predictive Modeling (Linear Regression)

## Tech Stack
- React / Vite
- Plotly
- Python (for analytics)

## How to Run
```bash
npm install
npm run dev
